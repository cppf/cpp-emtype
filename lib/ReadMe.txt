emType is a type definition and conversion library for emdb library. It has been developed for
for being used in the development of the internet of things for AVR processors. For information,
on its usage, please visit: https://github.com/wolfram77/embd.

For use with any AVR C complier (this includes AVRStudio, WinAVR, Arduino, Waspmote), simply copy
the contents of this folder to the source folder (or the sketch folder) and include the file: 
"embd.h" in the top of your program. Then make appropriate modifications to the "embd.h" file
(info given in the file).
